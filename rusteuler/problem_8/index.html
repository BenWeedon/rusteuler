Find the thirteen adjacent digits in the 1000-digit number that have the greatest product. What is the value of this product?

This solution relies mainly on the rust standard library. We create an array where each element is one of the digits. We then call the windows method on this array, which returns an iterator over windows of size 13. Finally, for each window we get the product (being careful to use u64 to not overflow) and call max on the full window iterator.
